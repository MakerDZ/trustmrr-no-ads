# TrustMRR Ads Remover

![Demo](demo.gif)

## How to Use

1. Go to `chrome://extensions/` and enable **Developer mode**
2. `git clone` this repo
3. Click **Load unpacked** and select the folder

That's it! Visit TrustMRR and toggle the extension to remove ads.
